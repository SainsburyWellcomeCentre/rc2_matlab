1. To start-up, 
   on Vis stim PC, run
	>> VisualStimuli_startup();   % configure visual stimuli
   on NIDAQ PC, run
	>> RC2_DoubleRotation_startup();   % start RC2 interface

2. To run experiment, 
   on RC2 interface, 
	select protocol file on the Experiment panel in folder 'Protocols'
	input Animal ID# & Session#
	click 'Start Experiment' button on the Experiment panel

3. If there are connection issues try the following on vis stim PC.
	>> visstim.reset_tcp_servers();
	>> visstim.is_running = false;

4. To exit, 
   on NIDAQ PC, run
	>> RC2_DoubleRotation_shutdown();
   on Vis stim PC, run
	>> clear;

If IP errors occur, modify the IP address in both start-up files.

If erros occur during experiment, run
	>> gui.experiment_seq.cleanup()
   to terminate experiment.
